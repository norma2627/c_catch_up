# C operators
C演算子(C++組み込み演算子のサブセット)


演算子の種類：
- 単項式
 - 単項演算子
 - 単項演算子に続くオペランド
- sizeofか_Alignofキーワード
- キーワードに続く式

式:
- 変数の名前
- キャスト式


## 単項演算子
シンボル 	名前
- ~ ! 	　　否定演算子と補数演算子
* & 	　　間接演算子とアドレス演算子
_Alignof 	アラインメント演算子 (C11 以降)
sizeof 	　　サイズ演算子
+ 	　　　　単項プラス演算子
++ -- 	　　単項インクリメント演算子と単項デクリメント演算子


## 二項演算子
シンボル 	　　　　名前
* / % 	　　　　　　乗算演算子
+ - 	　　　　　　加法演算子
<< >> 	　　　　　　シフト演算子
< > <= >= == != 	関係演算子
& | ^ 	　　　　　　ビットごとの演算子
&& || 	　　　　　　論理演算子
, 	　　　　　　　　順次評価演算子


演算子を含む式には代入式が含まれる。単項代入演算子、二項代入演算子が使用される。
