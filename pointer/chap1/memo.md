# ポインタとは

## オブジェクトとアドレス
- 変数: ポインタと深い関係にある
- 箱: 型(type)
  e.g.
  - int型
  - int型の変数m
  - int型の変数n
  - double型
  - double型の変数x
  - double型の変数y
- 変数: 型に基づいて作られた実態
- コンピュータのメモリにデータを文字できる
- 変数をオブジェクトと呼ぶ
  
場所を表す: 番地
番地のこと: アドレス

型の種類
- オブジェクト型
- 関数型
- 不完全型
